📊 Dashboard

📊 ¿Cómo usar el Dashboard?

El Dashboard es tu panel principal. Aquí ves un resumen de todo tu negocio de un vistazo.

  • Tarjetas superiores: Total de pedidos, ventas totales, ticket promedio y clientes únicos
  • Filtros de fecha: Usa los filtros de arriba para ver estadísticas de un mes o día específico
  • Gráficos: Muestran tus productos más vendidos, distribución por tipo de entrega y tendencias
  • Estado de Resultados: Resumen financiero con ventas brutas, descuentos, envíos y ventas netas
💡 Tip: Usa el botón "Exportar Excel" para descargar un reporte completo con todas las estadísticas.

📋 Pedidos

📋 ¿Cómo usar Pedidos?

Aquí gestionas todos los pedidos de tus clientes.

  • Filtros: Busca por nombre, número, teléfono o filtra por estado y fecha
  • Cambiar estado: Haz clic en un pedido → selecciona el nuevo estado (Pendiente → Confirmado → Preparando → Listo → Entregado)
  • WhatsApp automático: Al cambiar el estado, se genera un mensaje listo para enviar al cliente
  • Cancelar: Marca el pedido como "Cancelado" (se mantiene en el historial)
  • Eliminar: Borra permanentemente el pedido de la base de datos (no se puede deshacer)
💡 Tip: Los pedidos Pendientes se cancelan automáticamente después de 24 horas sin confirmar.

🍪 Productos

🍪 ¿Cómo usar Productos?

Aquí administras el catálogo de lo que vendes.

  • Agregar: Haz clic en "+ Agregar" para crear un nuevo producto
  • Editar: Haz clic en el ícono de lápiz para modificar nombre, precio o imagen
  • Eliminar: Haz clic en el ícono de basura para quitar un producto
  • Imagen: Sube una foto clara del producto (se guarda automáticamente)
💡 Tip: Usa fotos de buena calidad y descripciones atractivas para vender más.
🎉 Ofertas y Promociones

🎉 ¿Cómo usar Promociones?

Crea ofertas automáticas que se aplican a los pedidos de tus clientes.

  • Descuento %: Ej: 10% de descuento en todos los pedidos
  • Descuento fijo: Ej: RD$50 de descuento
  • Envío gratis: Elimina el costo de delivery/envío
  • Lleva X paga Y: Ej: Lleva 3 paga 2 (BOGO)
  • Compra mínima: El descuento solo aplica si el cliente gasta cierto monto
  • Fechas y horas: Programa la promo para que inicie y termine automáticamente
  • Límite de usos: Define cuántas veces se puede usar (ej: primeros 50 clientes)
💡 Tip: Las promociones se aplican automáticamente cuando el cliente hace el pedido. No necesitas hacer nada manual.
📋 Tipos de promoción disponibles:
🏷️ Banner - Solo informativo 🚚 Free Delivery - Envío local gratis 📦 Free Envío - Envío nacional gratis 🎁 Todo Gratis - Todos los envíos gratis % Descuento % - Porcentaje de descuento 💵 Descuento Fijo - Cantidad fija RD$ 🛒 2x1 - Compra 2 lleva 1 gratis ⭐ Cliente Nuevo - Solo para nuevos 📱 Por WhatsApp - Solo pedidos por WhatsApp 🎂 Pedidos +X - Según cantidad

⚙️ Configuración

Personaliza tu tienda y sistema

⚙️ ¿Cómo usar Configuración?

Aquí ajustas toda la tienda. Cambios con efecto inmediato.

  • Abrir/Cerrar tienda: El switch de arriba. Si la tienda está cerrada, los clientes no pueden hacer pedidos
  • Dirección de recogida: La dirección donde los clientes van a buscar sus pedidos
  • Precios de envío: Define cuánto cobras por delivery y envío nacional
  • Teléfono de la tienda: Se usa para los mensajes de WhatsApp
  • Cuentas bancarias: Los datos donde los clientes hacen las transferencias
  • Colores: Personaliza los colores de tu tienda online
  • Email: Configura el correo para recibir notificaciones automáticas de pedidos
  • Mensajes: Personaliza los mensajes que se envían al cliente en cada estado
  • Administradores: Agrega o cambia contraseñas de otros administradores
💡 Tip: Para Gmail, usa una "Contraseña de aplicación" (no tu contraseña normal). Busca "contraseña de aplicación Gmail" en Google.

👥 Gestión de Clientes

👥 ¿Cómo usar Clientes?

Tu base de datos de clientes. Se llena automáticamente cuando alguien hace un pedido.

  • Registro automático: Cada vez que alguien hace pedido, se crea o actualiza su registro
  • Buscar: Usa la barra de búsqueda para encontrar por nombre o teléfono
  • Ver historial: Cada cliente muestra cuántos pedidos ha hecho y cuánto ha gastado
  • Editar: Actualiza datos del cliente (dirección, sector, notas)
  • Desactivar: Si un cliente causa problemas, puedes desactivarlo (no podrá hacer pedidos)
  • Exportar: Descarga reportes de tus clientes en Excel
💡 Tip: Los clientes con email registrado recibirán notificaciones automáticas cuando hagan un pedido.
